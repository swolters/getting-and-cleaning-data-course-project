# Code Book

This code book lists all data fields in `tidy.txt`.

## Identifiers

* subject - identifier of the test subject
* activity - activity type when measurements were taken

## Activity Types

* 1 WALKING: subject was walking during the test
* 
2 WALKING_UPSTAIRS
: subject was walking up a staircase during the test
* 3 WALKING_DOWNSTAIRS: subject was walking down a staircase during the test
* 
4 SITTING: subject was sitting during the test
* 
5 STANDING: subject was standing during the test
* 
6 LAYING
: subject was laying down during the test
